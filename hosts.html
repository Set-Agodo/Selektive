Design Your Gathering

  • Log In to the Host Dashboard: Access your personalized dashboard to begin creating your event and define the event's parameters

  • Define Logistics: Use the basic setup fields to define the physical details of your event:
    • Name & Theme: Choose an engaging title and theme.
    • Time & Location: Set the date, time, and physical or virtual location.
    • Capacity Limit: Set the maximum number of confirmed guests. This cannot be changed once voting closes.

  • Set Voting Rules: Determine how your guests will be chosen:

    • Voting Window: Determine how long the nominee pool has to cast their votes (e.g., 48 hours, 7 days).
    • Requirements: Specify any criteria voters must meet (e.g., minimum profile completion, previous attendance history).
